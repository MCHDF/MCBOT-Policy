MCBOT Privacy Policy

Last updated: 2026-06-24

1. Overview

This Privacy Policy explains how MCBOT handles data when used in Discord servers.

2. Data We May Collect

MCBOT may collect and process limited Discord data required to operate bot features, including:

  • Discord user IDs
  • Discord server IDs
  • Discord channel IDs
  • Command usage data
  • Bot configuration data set by server administrators
  • Error logs needed to diagnose and fix bot issues

3. How We Use Data

Collected data is used to provide bot functionality, store server-specific settings, process commands, prevent abuse, and troubleshoot errors.

4. Message Content

MCBOT does not intentionally store message content unless a feature explicitly requires it for operation, moderation, logging, configuration, or troubleshooting.

5. Data Sharing

We do not sell user data. Data is not shared with third parties except when required to operate the service, comply with legal obligations, protect users, or protect the bot and its infrastructure.

6. Data Retention

Configuration and operational data may be retained for as long as needed to provide MCBOT features. Temporary logs may be deleted periodically when no longer needed.

7. Data Removal

Users or server administrators may request data removal by contacting Discord @mchdf or emailing support@mchdf.moe. Some data may need to be retained when required for security, abuse prevention, or legal compliance.

8. Security

Reasonable measures are used to protect stored data. No online service can guarantee absolute security.

9. Changes

This policy may be updated from time to time. The latest version will be available at this page.

10. Contact

For privacy questions or requests, contact Discord @mchdf or email support@mchdf.moe.
